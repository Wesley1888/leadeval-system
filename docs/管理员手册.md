# 中煤平朔集团有限公司中层管理人员考核系统管理员手册

## 1. 管理员概述

### 1.1 管理员职责
管理员负责系统的整体管理，包括考核人员管理、考核任务配置、数据统计分析、系统维护等工作。

### 1.2 管理员权限
- 人员信息管理
- 部门信息管理
- 考核任务管理
- 权重配置管理
- 数据统计分析
- 系统设置维护

## 2. 登录管理

### 2.1 管理员登录
1. 访问系统登录页面
2. 点击"管理员登录"按钮
3. 输入管理员账号和密码
4. 登录成功后进入管理后台

### 2.2 安全要求
- 定期更换密码
- 不在公共设备上登录
- 及时退出登录
- 保护账号安全

## 3. 人员管理

### 3.1 被考核人员管理
- **添加人员**：录入被考核人员基本信息
- **编辑信息**：修改人员信息
- **删除人员**：删除无效人员记录
- **批量导入**：支持Excel批量导入

### 3.2 人员信息字段
- 姓名
- 性别
- 出生年月
- 学历
- 职称
- 政治面貌
- 现任职务
- 任职时间
- 分管工作
- 所属部门

### 3.3 考核码管理
- **生成考核码**：为参评人员生成8位考核码
- **分配考核码**：将考核码分配给相应人员
- **考核码状态**：查看考核码使用状态
- **重置考核码**：重置已使用的考核码

## 4. 部门管理

### 4.1 部门信息维护
- **部门添加**：新增部门信息
- **部门编辑**：修改部门信息
- **部门删除**：删除无效部门
- **部门层级**：维护部门层级关系

### 4.2 部门分类
- 公司职能管理部门
- 二级单位
- 独资公司
- 参控股公司
- 托管单位

## 5. 考核任务管理

### 5.1 创建考核任务
- **任务名称**：设置考核任务名称
- **考核年度**：选择考核年度
- **考核范围**：确定考核人员范围
- **考核时间**：设置考核开始和结束时间
- **权重配置**：设置各考核人权重

### 5.2 权重配置
根据考核办法要求配置权重：

#### 5.2.1 公司班子成员（权重50%）
- 公司总经理：20%
- 党委书记：20%
- 分管领导：15%
- 其他班子成员：45%

#### 5.2.2 公司中层管理人员（权重25%）

#### 5.2.3 单位班子成员相互测评及基层管理人员、职工代表民主测评（权重25%）
- 单位班子成员相互测评：最高30%
- 基层管理人员、职工代表测评：不低于70%

### 5.3 任务状态管理
- **未开始**：考核任务创建但未开始
- **进行中**：考核正在进行
- **已完成**：考核已完成

## 6. 数据统计分析

### 6.1 考核结果统计
- **个人得分**：查看个人详细得分
- **部门统计**：按部门统计考核结果
- **等级分布**：统计各等级人员分布
- **权重分析**：分析不同权重对结果的影响

### 6.2 考核等级标准
- **优秀**：得分90分以上
- **良好**：得分75-89分
- **一般**：得分60-74分
- **较差**：得分59分以下

### 6.3 数据导出
- **Excel导出**：导出考核结果到Excel
- **PDF报告**：生成PDF格式考核报告
- **图表导出**：导出统计图表
- **数据备份**：定期备份考核数据

## 7. 系统管理

### 7.1 用户管理
- **管理员账号**：管理管理员账号
- **权限分配**：分配不同权限
- **登录日志**：查看登录记录
- **操作日志**：记录操作历史

### 7.2 系统设置
- **系统参数**：配置系统参数
- **考核指标**：维护考核指标
- **评分规则**：设置评分规则
- **通知设置**：配置系统通知

### 7.3 数据维护
- **数据备份**：定期备份数据
- **数据恢复**：恢复备份数据
- **数据清理**：清理无效数据
- **数据校验**：校验数据完整性

## 8. 考核组织实施

### 8.1 考核前准备
1. **制定考核方案**：根据考核办法制定年度考核工作方案
2. **确定考核范围**：明确被考核人员范围
3. **配置考核权重**：设置各考核人权重
4. **生成考核码**：为参评人员生成考核码
5. **发布考核通知**：通知相关人员参加考核

### 8.2 考核过程管理
1. **监控考核进度**：实时监控考核完成情况
2. **处理异常情况**：处理考核过程中的异常
3. **数据实时统计**：实时统计考核数据
4. **确保数据安全**：保障考核数据安全

### 8.3 考核结果处理
1. **结果统计汇总**：汇总各层次打分结果
2. **形成考核报告**：生成考核结果报告
3. **结果反馈**：向被考核单位反馈结果
4. **结果应用**：将结果应用于人事管理

## 9. 常见问题处理

### 9.1 技术问题
- **系统无法访问**：检查网络和服务器状态
- **数据丢失**：从备份恢复数据
- **性能问题**：优化系统配置
- **安全漏洞**：及时修复安全漏洞

### 9.2 业务问题
- **考核码无效**：重新生成考核码
- **权重配置错误**：修正权重配置
- **数据统计异常**：检查数据完整性
- **用户投诉**：及时处理用户反馈

## 10. 联系支持

### 10.1 技术支持
- **技术团队**：系统开发维护团队
- **运维团队**：系统运维团队
- **安全团队**：系统安全团队

### 10.2 业务支持
- **人力资源部**：考核业务指导
- **组织部门**：组织管理指导
- **纪检监察**：廉政建设指导

---

**重要提醒**：管理员应严格按照考核办法要求组织实施考核工作，确保考核过程的公平、公正、公开。
